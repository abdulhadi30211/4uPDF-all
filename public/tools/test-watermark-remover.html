💡 Old Error Messages (Not Helpful):

✨ New Error Messages (Actually Useful):

Invalid PDF file - missing PDF header. File may be corrupted or not a PDF.

Failed to parse PDF: [specific error]. The file may be corrupted.

Common Errors & Solutions:

❌ "Invalid PDF file - missing PDF header"
Cause: File is not actually a PDF or is corrupted
Solution: Use a real PDF file, try downloading again
❌ "Failed to parse PDF document"
Cause: PDF structure is damaged or encrypted
Solution: Remove password protection, use different PDF
❌ "File is too small to be a valid PDF"
Cause: Empty or nearly empty file
Solution: Check file size, ensure complete upload

📊 Removal Modes Explained

🤖

Auto Detect

Scans PDF text for common watermark patterns like "DRAFT", "CONFIDENTIAL", "SAMPLE". Automatically covers detected text with white rectangles. Also checks corners and center.

✓ Best for: Most cases (recommended first choice)

✋

Manual Select

Covers 5 standard watermark positions: bottom-right, bottom-left, center, very bottom-right, and top-left corners. No detection, just coverage.

✓ Best for: Predictable watermark placements

⚡

Batch Process

Applies light overall coverage to fade watermarks. Focuses on edges where watermarks commonly appear. Subtle treatment with low opacity.

✓ Best for: Multiple files, light watermarks

🎯

Precision Mode

Advanced detection for rotated text (common in diagonal watermarks). Finds semi-transparent text. Uses sophisticated pattern analysis.

✓ Best for: Difficult, rotated, or transparent watermarks
