🛠️ Troubleshooting

If signature doesn't appear in downloaded PDF:
  1. Check browser console (F12) for error messages
  2. Verify signature was created (check console for "Signature field added")
  3. Make sure signature field is visible on PDF preview
  4. Try a different signature method (draw/type/upload)
  5. Use a simpler/smaller PDF for testing
  6. Clear browser cache and reload
  7. Try in a different browser (Chrome recommended)
